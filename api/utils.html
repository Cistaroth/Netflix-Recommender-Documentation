property best_row_strategy: str#

The strategy to use for selecting the best row.

The possible values are ‘max’ and ‘min’. ‘max’ will select the row with the highest score, while ‘min’ will select the row with the lowest score.

The strategy to use for selecting the best row.

property knn_metrics: list[str]#

The metrics to use for the K-Nearest Neighbors algorithm.

The possible values are the metrics available in the scikit-learn library, such as ‘minkowski’, ‘cosine’, ‘manhattan’, etc.

Returns:

The metrics to use for the K-Nearest Neighbors algorithm.

Return type:

list[str]

property knn_n_neighbours: list[int] | range#

The number of nearest neighbours to consider when using the K-Nearest Neighbors algorithm.

This can be either a list of integers or a range object. If a list of integers is provided, the model will be trained with each of the provided values. If a range object is provided, the model will be trained with each of the values in the range.

Returns:

The number of nearest neighbours to consider.

Return type:

list[int] | range

property knn_weights: list[Literal['uniform', 'distance']]#

The weights to use for the K-Nearest Neighbors algorithm.

The possible values are ‘uniform’ and ‘distance’. ‘uniform’ will assign equal weights to all neighbours, while ‘distance’ will assign weights to the neighbours based on their distance.

Returns:

The weights to use for the K-Nearest Neighbors algorithm.

Return type:

list[Literal[“uniform”, “distance”]]

property pca_n_components: list[float] | range#

The number of components to retain after applying PCA.

This can be either a list of floats between 0 and 1, or a range object. If a list of floats is provided, the number of components to retain will be the length of the list. If a range object is provided, the number of components to retain will be the length of the range.

Returns:

The number of components to retain after applying PCA.

Return type:

list[float] | range

property product: list[tuple]#

Returns the product of the hyperparameters.

This is a list of tuples, where each tuple contains one value from each of the hyperparameters. The length of the returned list is the product of the lengths of the hyperparameters.

Returns:

The product of the hyperparameters.

Return type:

list[tuple]

property validation_metric: str#

The validation metric for cross-validation.

This is the metric used to evaluate the model during cross-validation. The possible values are the same as those for the ‘scoring’ parameter in sklearn.model_selection.cross_val_score.

Returns:

The validation metric.

str
